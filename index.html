What is this?

Discord+ is a theme for Discord. It's both sleek and customizable, without getting in your way. Its top priority is balancing between looks and performance.

All you need to install my theme is BetterDiscord. If you already have it, you're ready to install my theme!

All you need to install my theme is a supported computer with Discord and BetterDiscord installed.

User customization

This theme is very customizable. The variables at the top of the file allow you to change things like the background image, the accent color, the roundness of the UI, the fonts, and even the backdrop behind elements. Just open the file up in a text editor, and you're on your way.
